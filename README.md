# LAMBO: THE GAME

Made for my cousins' bday. For context: they have a bunny named Lambo because he moves really fast. This is a game where Lambo rides an actual lambo.
This game was made entirely in Godot. No AI was used in the creation of this project!

Play at https://pxlinspace.itch.io/lambo
